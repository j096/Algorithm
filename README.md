# Algorithm
인프런 알고리즘 강의를 들으며 코딩한 내용 정리

## basic
코드 구현력을 기르기 위한 단순 연산 문제

## array_search
문자열, 1차원/2차원 배열 탐색

## binary_search
이분 탐색, 결정 알고리즘

## greedy_algorithm
그리디 알고리즘

## datastructure
스택, 큐, 해시, 힙

## recursive
재귀함수를 이용하여 구현하는 알고리즘<br>
깊이우선탐색(bfs), 넓이우선탐색(dfs)

## dynamic_programming
동적계획법, 냅색 알고리즘
